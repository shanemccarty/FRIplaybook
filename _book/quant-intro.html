Part III: Analyzing Quantitative Data {.unnumbered}

Section Editors: Shane McCarty, PhD & Zihan Hei

This section of the FRI Public Health Playbook focuses on quantitative data analysis methods essential for public health research. We cover the complete workflow from data import through advanced statistical analysis and visualization.

1.1 What Youâ€™ll Learn

In this section, you will:

  • Master data import techniques for various file formats commonly used in public health research

  • Learn to clean, transform, and prepare data for analysis

  • Apply descriptive and inferential statistics to public health datasets

  • Create publication-quality visualizations

  • Interpret and communicate quantitative findings effectively

1.2 Section Contents

This section includes the following chapters:

  1. Importing and Visualizing NHANES Data - Learn the fundamentals of data import and exploration using the National Health and Nutrition Examination Survey dataset

  2. Additional chapters on statistical analysis (coming soon)

  3. Advanced visualization techniques (coming soon)

1.3 Prerequisites

Before beginning this section, you should have:

  • Basic familiarity with R or another statistical programming language

  • Understanding of fundamental statistical concepts

  • Access to a web browser (for WebR interactive examples)

1.4 About the Section Editors

Shane McCarty, PhD is the lead author of the FRI Public Health Playbook and specializes in quantitative methods for public health research.

Zihan Hei is a contributing researcher focusing on data visualization and statistical analysis in public health contexts.
