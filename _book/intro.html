Students in the FRI Public Health research stream at Binghamton University explore a variety of biopsychosocial factors affecting human physical and mental health. This playbook includes a variety of “plays” (R code) that students in the quantitative data analysis track can use for their team-based research projects.

1.1 The Data Analysis Workflow

This playbook follows the data science workflow in R for Data Science (2e) (Hadley, 2023) to help researchers examine the prevalence of disease and wellbeing indicators, describe health patterns, identify associations between (risk/protective/promotive) determinants and (positive/negative) health outcomes, and produce a reproducible scientific report with R and Quarto.

  1. Import - Load data from various sources, such as standard (.csv), Excel (.xlsx), and SPSS (.sav)
  2. Tidy - Organize data into a consistent, analysis-ready structure
  3. Transform - Clean, recode, and create new variables using dplyr
  4. Visualize - Create informative graphs and charts with ggplot2
  5. Model - Apply statistical methods to answer research questions
  6. Communicate - Generate reproducible reports with Quarto

Each section builds on previous concepts, taking you from raw data files to polished, publication-ready analyses.

1.1.1 What Makes This Book Different?

  • Interactive Learning: Some code examples can be copy and pasted into your .R script, .qmd markdown report file, or run in the console. Some chapters include code examples that run directly in your browser using WebR—no software installation required
  • Real Public Health Data: Examples use datasets on insurance claims from kaggle, oral health and diabetes (cohort 10), health beliefs (cohort 11), and related surveys
  • Complete Workflow: Covers every step from raw data to publication-ready reports
  • Practical Focus: Emphasizes “plays” (R code chunks) you’ll actually use in public health research and practice

1.2 What You’ll Learn

By the end of this book, you will be able to:

  • Import and manage data from multiple file formats commonly used in public health (CSV, Excel, SPSS, SAS)
  • Clean and prepare datasets for analysis using tidyverse tools
  • Transform variables by recoding, creating dummy variables, and calculating composite scores
  • Create publication-quality visualizations that effectively communicate health data
  • Apply statistical models including regression, paired tests, and other inferential methods
  • Generate reproducible reports that integrate code, results, and narrative
  • Follow best practices for transparent, reproducible public health research
